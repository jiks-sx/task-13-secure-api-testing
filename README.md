# Secure API Testing & Authorization Validation

## Objective
Test REST APIs for authentication and authorization weaknesses using Postman or cURL.

## Tools Used
- Postman
- cURL
- Test REST API endpoints

## Workflow
1. Configure API endpoint in Postman
2. Test authentication behavior
3. Remove authentication headers
4. Test authorization access
5. Send malformed inputs
6. Test rate limiting
7. Analyze HTTP responses
8. Document vulnerabilities
